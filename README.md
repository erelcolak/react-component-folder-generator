# React Component with Folder with File Tree Generator

Generates a React component folder context menu

**Usage:**

- Right click any folder!
- Type your component name

```
targetFolder/ (right click)
└── SampleComponent/
    ├── index.ts
    ├── SampleComponent.tsx
    ├── SampleComponent.types.ts
    └── SampleComponent.module.scss
```
